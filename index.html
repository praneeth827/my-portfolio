School Management System With Integrated Learning Platform

This project aims to develop a comprehensive school management system. It integrates features like attendance tracking, marks management, student data storage, and hosting educational videos aligned with the syllabus by using HTML, CSS, JS and MySQL with PHP.

Emergency Response Resource Allocation in Hospitals Using Bipartite Graph Matching

This project is done based on Bipartite graph with the Hopcroft-Karp algorithm with BFS and DFS which is useful for better matching by using augmenting paths. By pairing available staff with emergency cases efficiently, this approach enhances response times and overall effectiveness in managing urgent medical situations. Tools used: GOOGLE COLAB, MATPLOTLIB, NUMPY.

Universal Warranty and Bill Tracking System

Jewellery Shop Monitoring System

This project is a Jewellery Shop Monitoring System designed to simplify online jewellery shopping. It allows users to view, select, and purchase jewellery, while managing product details like name, quantity, weight, and images. The system calculates billing with GST and tracks payment status. It also records lifetime revenue for the shop, helping in daily sales analysis. Built using Python with a GUI in Tkinter, the project uses functions, loops, and modules to ensure smooth user experience and backend processing.
